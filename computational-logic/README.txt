src/ - contains the source
exe/ - contains the executable

Running:
    If you have python installed run the script in the 'src' directory:
        python main.py

    Or run the executable in the 'exe' directory':
        main.exe


Compiling:
    Requirements:
        You must have installed python version 2.6 and py2exe

    The actual compilation procedure(in the 'src' directory):
        python setup.py py2exe

    The above command will compile the project and the exe files will be in 'dist' directory
